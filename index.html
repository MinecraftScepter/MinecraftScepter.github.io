資訊大集合

開服需要的各種資訊，全部在一個畫面呈現出來，不論是需要哪個 IP，還是系統效能，又或是得知 Java 版本，只需要看一眼！

伺服器管理

針對玩 Vanilla 的玩家有特別優待外，要整合如 Forge, Fabric, Spigot ...等伺服器也都支援，除了管理功能，也有人性化設定檔介面、候選指令、強制停止以及更多進階功能可供探索～
推薦給...
開服萌新
透過軟體開啟，大大降低開服出錯的可能，剩下的問題，肯定就是連線設定的問題！
略有經驗
伺服器除了官方還支援 Fabric, Forge, Spigot ...等種類，還可以迅速切換與管理不同伺服器，另有指令提示與進階參數...等功能等著使用！
直播主
有不止一張地圖想與觀眾們或其他主播遊玩？預先準備好後，不論伺服器種類或地圖，只要點幾下就可以切換！
地圖作者
魔方支援將地圖連同伺服器打包成一個極小的安裝包，可以讓其他玩家方便管理與分享，非常有利於地圖推廣！
開發者
魔方是開源軟體，要是有開發知識，歡迎點右上角 Github 來討論與回饋！或是根據自己的需要作出一個特別版～
