What is 127.0.0.1? Can that be used to connect?

This is a IP represent to yourself. It's mostly used for testing If server is actually working.

How to connect to friend?

There is three way to connect to friend, Virtual Lan, Router, Ngrok. Hamachi is the common way to everyone, and Configure router to use port forwarding on 25565 is the second way. The final way is Ngrok, you could use this without open port.

What did Minecraft Scepter do? Why I need to use It?

This is Chinyun's dream in childhood, create a server headlessly, manage multiple server, and automating any part with friendly steps! You might not need to use Minecraft Scepter with heavy server. It's currently support, Forge and Vanilla, so you should consider If this fit your needs.

Could existed server migrate in Minecraft Scepter?

Why not? Just copy in servers folder beside exe, If there is no such a folder, make one. And change the original server executable to server.jar, and make a empty dmc with vanilla type selected, that's it.

Support Minecraft (MCPE)

There is no such a official way to open MCPE, and the only way to open PE server is used third party, since I didn't realize any license about MCPE, so the support is not planed.

Crash?

Did It support X.X.X version

A lot of IP, which do I use?

Since my video is Chinese, I'll expain here, If your router's port forward method is configured properly, public IP is the best choice. The second way, use VLAN like hamachi, so you don't need to know anything listed. The final killer approach is ngrok, after sign up, get a auth token and paste in, that's It!

Did XYZ Big fat mod work?

You could set java configuration or any server configuration in the Scepter now, so the openable should count on your machine.

When add Mod B, my server crash, but A is working?

This is not the fault of scepter, It's a generally plugin/mod confliction or unstability from mod, you should report or ask to forge or mod author.

Making a installer and follow the steps, why my installer doesn't work?

You might download wrong forge, such as Universal verison, or not using direct download link, It's a link that could be download file directly without REDIRECT.
